Esta API esta hecha para calcular

de grados °C a grados °F

y vice-versa

Esta API puedes ser utilizada mediante axios por javascript

usando la URL "temperature/CelsiusTofahrenheit/20" el numero 20 es el valor que se quiere consultar osea 20 °C a °F. Por otro lado para consultar de Fahrenheit a Celsius se debe utilizar la URL 'temperature/FahrenheitToCelsius/numero' igual que en el anterior, "numero" se reemplaza por el numero que se desea consultar se supone que son °F y va a consultar los grados °C
